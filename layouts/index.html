What is EQE?

Event-Centric Query Expansion (EQE) is a real-time and efficient query expansion framework designed for timely search scenarios. It is proposed by researchers from Tencent QQ Browser Search and deployed in a real production environment.

For more details, please refer to our ACL Industry 2023 paper:

(Zhang2023EventCentricQE)

One More Thing

We disclose an annotated test dataset for Event Reformulation, which is called Title2EventPhrase. This open dataset features a large-scale collection of human-annotated Chinese titles and includes 41,000 <headline, event> pairs. Further details can be found at

title2eventphrase
