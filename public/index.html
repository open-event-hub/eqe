Event-Centric Query Expansion in Web Search

What is EQE?

Event-Centric Query Expansion (EQE) is a real-time and efficient query expansion framework designed for timely search scenarios. It is proposed by researchers from Tencent QQ Browser Search and deployed in a real production environment.
